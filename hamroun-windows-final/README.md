# Hamroun - Langage de Programmation Français

##  PROBLÈME DE COMPATIBILITÉ RÉSOLU

### Si Windows dit "Cette application ne peut pas s'exécuter":

**SOLUTION SIMPLE:**
1. Essayez d'abord hamroun-32bit.exe (plus compatible)
2. Si ça ne marche pas, clic droit  Propriétés  Décocher "Débloquer"
3. Ou cliquez "Plus d'infos" puis "Exécuter quand même"

##  Contenu

- **hamroun-32bit.exe** - Version 32-bit (RECOMMANDÉE - plus compatible)
- **hamroun-64bit.exe** - Version 64-bit 
- **simple.hamroun** - Premier programme
- **demo.hamroun** - Démonstration complète

##  Test rapide

`
hamroun-32bit.exe simple.hamroun
`
