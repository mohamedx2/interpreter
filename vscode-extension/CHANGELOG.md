# Change Log

All notable changes to the "hamroun-language" extension will be documented in this file.

## [1.0.0] - 2025-12-18

### Added
- Initial release of Hamroun Language Support
- Syntax highlighting for all French keywords
- 15+ code snippets for common constructs
- Language configuration with:
  - Auto-closing brackets and quotes
  - Comment toggling
  - Smart indentation
  - Code folding
- Professional icon with French flag colors
- Comprehensive documentation

### Keywords Supported
- Control flow: `SI`, `ALORS`, `SINON`, `FIN`, `TANT_QUE`, `FAIRE`
- Loops: `BOUCLE`, `DE`, `A`
- Functions: `FONCTION`, `PROCEDURE`, `RETOURNER`
- Operators: `EGAL`, `DIFFERENT`, `PLUS_GRAND`, `PLUS_PETIT`
- I/O: `AFFICHER`, `LIRE`
- Logic: `ET`, `OU`, `NON`
- Types: `ENTIER`, `REEL`, `TEXTE`, `BOOLEEN`, `TABLEAU`, `LISTE`
